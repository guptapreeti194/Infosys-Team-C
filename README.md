This repository contains the collaborative work of Team C developed during the Infosys Springboard Virtual Internship 6.0. The project, ClauseEase AI, is designed as an offline, privacy-centric document assistant that leverages locally hosted AI models to process and understand documents without relying on cloud services.

Each team member has contributed their own implementation and ideas, making this repository a shared workspace that reflects our combined efforts, learning, and problem-solving approach.
